# Valentine's Day Proposal Page

A lightweight, interactive Valentine's Day webpage designed to ask "Will you be my Valentine?".

## Features

- **Interactive Buttons:**
  - Clicking "No" makes the "Yes" button larger and triggers a sad emoji shower.
  - Clicking "Yes" triggers a celebration with happy emojis and a success message.
- **Responsive Design:** Looks great on mobile and desktop.
- **Single File:** All code (HTML, CSS, JS) is contained in `valentine.html` for easy sharing.

## How to Use

Simply double-click `valentine.html` to open it in your web browser.

## Customization

You can edit `valentine.html` in any text editor to change:
- The placeholder image URL.
- The colors in the CSS section.
- The text messages or emojis in the JavaScript section.
